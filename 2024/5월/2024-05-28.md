#### C++ 서버 강좌
```cpp
#include <iostream>
using namespace std;

// 분기문
int main(void)
{
	const int ROCK = 0;
	const int PAPER = 1;
	const int SCISSORS = 2;

	int input = ROCK;

	if (input == ROCK)
		cout << "rock" << endl;
	else if (input == PAPER)
		cout << "paper" << endl;
	else if (input == SCISSORS)
		cout << "scissors" << endl;
	else
		cout << "?" << endl;

	// switch - case
	switch (input)
	{
	case ROCK:
		cout << "rock" << endl;
		break;
	case PAPER:
		cout << "paper" << endl;
		break;
	case SCISSORS:
		cout << "scissors" << endl;
		break;
	default:
		cout << "?" << endl;
		break;
	}
}
```

- 마찬가지로 switch문 또한 compare -> jmp의 구조임을 알 수 있다.
- ![[Pasted image 20240528150522.png]]
- 즉, if문과 swich문은 기본적으로 성능상의 차이는 적다는걸 알 수 있다.
- switch문은 else-if문과 다르게 정수 형식만 처리가 가능하다. 
- C#의 경우 string 형식도 처리가 가능하나, C++는 int형식만 가능
